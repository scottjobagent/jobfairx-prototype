Copy by breakpoint

BreakpointCopy

Both link to the same destination — the employer landing page (same target as the existing "Learn how →" link).

Layout summary

  • Desktop: existing light-blue banner background and link styling stay. Only the copy changes — drops "in one day" and "Learn how" becomes "Learn more".
  • Mobile: single line, centered. Question + inline CTA + close. Removes the second-line "Learn how" link.
  • Vertical height (mobile): shrinks by roughly 18px (one line removed + tighter padding).
  • Dismiss behavior: X close on the right (unchanged on both breakpoints).
  • Star/sparkle icon: stays on desktop (existing brand element).

QA

  • Desktop banner fits on one line at 1024px width (and wider).
  • Mobile banner fits on one line at 360px (smallest common mobile width).
  • Tapping the text or "Learn more →" on either breakpoint opens the employer landing page.
  • X dismiss persists for the session — no regression from the existing behavior.
  • Contrast passes WCAG AA on both backgrounds (dark text on light blue, white text on dark navy).
  • The destination URL is unchanged from the existing "Learn how →" link.
