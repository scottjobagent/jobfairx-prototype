Mobile Banner — Job Seeker Side

Sticky banner copy + layout cleanup

The current employer cross-promotion banner is two lines, wordy, and competes visually with the page. Replacing with a single-line question + inline CTA. Mirrors the structure of the existing job seeker banner on the employer side.

Summary of changes

  • Copy: from "Fill your open roles in one day with a virtual hiring event." + "Learn how →" → to "Are you hiring? Learn more →".
  • Layout: two-line headline + secondary CTA → single line, inline CTA.
  • Vertical height: banner shrinks by roughly 18px (one line removed + tighter padding).
  • Behavior: entire banner text is tappable. Underlined "Learn more →" inherits the same link target.
  • Dismiss: X close on the right (unchanged).
  • Page it links to: the employer landing page (same target as the old "Learn how →").
