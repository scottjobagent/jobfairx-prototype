What's changing

    Implementation notes

    Key points

    • Only the "scheduled candidate interviews" line is changing. Job counts, prices, recruiter seats, and all other copy remain the same.
    • The interview count should be determined by the event type of the page being viewed.
    • Formula: Growth = Starter count × 3, Pro = Starter count × 6.
    • Layout, styling, button text, and card structure are not changing. This is a content-only update.
