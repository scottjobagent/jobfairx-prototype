Key points

  • Only the interviews and hires stats are changing. The "Pre-Registered Candidates" section stays exactly as-is.
  • Labels should render uppercase to match existing design: "AVG. INTERVIEWS PER JOB"
  • Starter package interview promises need to be updated per event type (no longer a flat 20+ across all types).
  • Growth package = Starter count × 3 (e.g., Entry Level = 72+, Healthcare = 45+)
  • Pro package = Starter count × 6 (e.g., Entry Level = 144+, Healthcare = 90+)
  • Layout, styling, colors, and positioning of the stats bar are not changing. This is a content-only update.
